# Response Patterns

## Action Codes

| Code | Meaning |
| --- | --- |
| accept-text | Revise wording only |
| accept-analysis | Add analysis or experiment |
| add-citation | Add or correct citation |
| revise-figure | Update figure, caption, or supplementary figure |
| soften-claim | Narrow claim strength or scope |
| explain-scope | Politely explain why request is outside scope |
| author-input-needed | Requires human decision or unavailable data |

## Tone Pattern

Thank the reviewer -> state action -> point to manuscript location -> explain evidence or boundary.

## Disagreement Pattern

We agree that [issue] is important. However, [reason/evidence]. We have therefore [limited revision], and we now state [boundary] in [location].

## Never Invent

- Page or line numbers.
- New experiments.
- New citations.
- Figure panels.
- Supplementary tables.
- Editor decisions.
