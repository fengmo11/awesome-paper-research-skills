# Package Checklist

## Manuscript

- Title.
- Abstract.
- Keywords if required.
- Main text sections.
- Methods or supplementary methods.
- Acknowledgements.
- Funding.
- Author contributions.
- Competing interests.
- Data availability.
- Code availability.

## Figures And Tables

- Every figure/table is cited.
- Captions define abbreviations.
- Panels are labelled consistently.
- Source data is available where required.
- Resolution/export format matches venue rules.

## References

- In-text citations match bibliography.
- Bibliography format matches venue.
- DOI/metadata are not fabricated.

## Final Files

- Main manuscript.
- Figures.
- Supplementary information.
- Source data.
- Cover letter.
- Response to reviewers if resubmission.
