# Paper Submission Gate

Use this skill before submission or preprint posting. It is the final quality gate for the paper package.

## Inputs

- Target venue or preprint server.
- Manuscript files.
- Figures, tables, supplementary files.
- References or BibTeX.
- Data/code availability statements.
- Author declarations and cover letter if available.

## Workflow

1. Identify the venue package requirements.
2. Check manuscript structure and required sections.
3. Check figure/table numbering, captions, resolution, and source traceability.
4. Check reference style and citation consistency.
5. Check data/code availability, ethics, conflicts, funding, and author contributions.
6. Check supplementary files and naming.
7. Check export formats: Markdown, LaTeX, DOCX, PDF, SVG, PNG, TIFF when needed.
8. Return a submission readiness report with blockers, warnings, and optional polish.

## Submission Gate

- No placeholder text remains.
- Every cited figure/table exists and is numbered correctly.
- Every reference cited in text appears in the bibliography.
- Data/code availability is present or explicitly not applicable.
- Declarations required by the venue are present.
- Final package files are named consistently.
