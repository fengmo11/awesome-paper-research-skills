# Hedging And Overclaim Control

| Risky wording | Safer alternative |
| --- | --- |
| prove | show / suggest / provide evidence that |
| conclusively demonstrate | demonstrate in this setting |
| first | to our knowledge |
| superior | improved relative to the tested baseline |
| universal | in this cohort / dataset / condition |
| causes | is associated with / may contribute to |
| fully solves | addresses one aspect of |

## Audit Questions

1. Does the claim exceed the actual experiment?
2. Is a causal verb used for correlational evidence?
3. Is novelty asserted without a literature boundary?
4. Does the sentence imply clinical, industrial, or universal generality without support?
