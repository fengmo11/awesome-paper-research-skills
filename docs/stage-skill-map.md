# Stage Skill Map

This map turns scattered paper-related skills into a composable workflow.

## 1. Idea Discovery

Useful source patterns:

- `AI-Research-SKILLs/21-research-ideation`
- AI-Scientist style hypothesis generation
- Open Coscientist style idea review/ranking

Expected artifacts:

- Research question list.
- Novelty and feasibility table.
- Risk register.
- Candidate experiment plan.

## 2. Literature Search

Useful source patterns:

- `academic-research-skills/deep-research`
- `RE-paper-writing/skills/literature-search`
- `openags/paper-search-mcp`
- `jkitchin/litdb`

Expected artifacts:

- Search query log.
- Inclusion and exclusion criteria.
- Source table.
- Related-work clusters.
- BibTeX candidates.

## 3. Experiment-To-Paper Bridge

Useful source patterns:

- Hermes `research-paper-writing`
- AutoResearchClaw experiment and analysis phases.
- `RE-paper-writing/skills/experiment-report-bridge`
- data-to-paper traceability patterns.

Expected artifacts:

- Experiment summary.
- Reproducibility checklist.
- Results table.
- Statistical test plan.
- Figure plan.

## 4. Paper Writing

Useful source patterns:

- `AI-Research-SKILLs/20-ml-paper-writing`
- `academic-research-skills/academic-paper`
- Hermes `research-paper-writing`
- OpenDraft's multi-agent writing split.

Expected artifacts:

- Section outline.
- Claim-evidence map.
- Full Markdown draft.
- Abstract and title candidates.
- Contribution statement.

## 5. Citation Verification

Useful source patterns:

- `RE-paper-writing/skills/citation-verification-gate`
- OpenDraft verified citation workflow.
- AutoCitation.
- refchecker.

Expected artifacts:

- Citation audit table.
- DOI/arXiv/CrossRef/OpenAlex metadata check.
- Unsupported-claim list.
- BibTeX repair suggestions.

## 6. LaTeX / Word Formatting

Useful source patterns:

- Hermes LaTeX compilation phase.
- OpenDraft PDF/Word/LaTeX export.
- `RE-paper-writing/skills/paper-compilation`
- Venue template checklists.

Expected artifacts:

- LaTeX source package.
- DOCX export.
- Figure/table placement checklist.
- Venue compliance checklist.

## 7. Review And Revision

Useful source patterns:

- `academic-research-skills/academic-paper-reviewer`
- `RE-paper-writing/skills/review-ensemble`
- `poldrack/ai-peer-review`
- AutoResearchClaw multi-agent peer review.

Expected artifacts:

- Reviewer panel report.
- Scorecard.
- Major/minor issue list.
- Revision plan.
- Response-to-reviewer draft.
