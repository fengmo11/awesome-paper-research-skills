# Publication Playbook

This is the suggested growth plan for this awesome list.

## Positioning

Use a clear promise:

> A curated map of open-source tools for AI-assisted research, from idea
> discovery to literature search, experiments, writing, citations, LaTeX/DOCX,
> and peer review.

## Launch Checklist

- Make the repo public.
- Add topics: `ai-research`, `academic-writing`, `ai-scientist`,
  `literature-review`, `latex`, `citations`, `research-agents`,
  `awesome-list`.
- Add the Awesome badge.
- Pin the repo on the GitHub profile.
- Post a short launch note on X, Reddit, Hacker News, and relevant Discords.
- Open issues for missing projects so contributors can help.

## Update Rhythm

- Weekly: add new projects and update categories.
- Monthly: update observed star/fork signals.
- Quarterly: add a trend report and "best new tools" section.

## Star Drivers

People star lists when they are:

- Useful in the first 30 seconds.
- Sorted by problem, not just dumped links.
- Honest about limitations.
- Easy to contribute to.
- Updated visibly.

## Monetization Angle

This repo can support:

- GitHub profile credibility.
- AI research assistant consulting.
- Paper workflow automation services.
- Citation/review/report generation micro-SaaS ideas.
- DataAnnotation/Mercor-style application proof of work.
